# Changelog

## 1.0

Initial public release of **Create: Gravlax**.

### Content

- Sugar Beet cultivation and processing line.
- Evaporation Basin and Laboratory Centrifuge.
- Sugar crystals, molasses and the Steffen sugar-recovery process.
- Waste Water, Lime Residue and Lime Water recovery loop.
- Mineral salt refining and culinary seasoning.
- Concentrated potions, essences, spices and milk-based remedies.
- Gravlax Salmon and optional Farmer's Delight food integration.
- Centrifuge operating audio and final project branding.

### Technical

- Minecraft 1.21.1 / NeoForge / Java 21.
- Namespace: `create_gravlax`.
- Original project code and original assets licensed under MIT.
- Third-party adapted assets and audio documented separately.
