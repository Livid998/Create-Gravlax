# Create: Gravlax

![Create: Gravlax logo](src/main/resources/create_gravlax_logo.png)

**Create: Gravlax** is a NeoForge addon for Create focused on culinary
industry and food alchemy.

It expands food processing into mechanical production chains: sugar beet
refining, molasses recovery through the Steffen process, mineral salt,
concentrated potions, extracted essences, seasonings, gravlax products and
optional Farmer's Delight integration.

## Features

- Grow and process **Sugar Beet** into juices, raw sugar, crystals and molasses.
- Recover additional sugar from molasses through the **Steffen process**.
- Reuse **Waste Water** and **Lime Residue** by preparing **Lime Water**.
- Refine mineral salt and use it in culinary processing.
- Operate the **Evaporation Basin** and **Laboratory Centrifuge**.
- Concentrate Create potion fluid into typed concentrated potions.
- Centrifuge concentrated potions into coloured **Essences**.
- Turn essences into culinary **Spices** that season compatible foods.
- Prepare **Gravlax Salmon**, milk-based remedies and potion-derived food effects.
- Use optional integration with **Farmer's Delight** recipes and ingredients.

## Building

This repository contains the Gradle project definition and source files.
Use **Java 21** and a compatible Gradle installation to build the mod:

```bash
gradle build
```

The project intentionally does not bundle local convenience compilation
scripts.
